# 🧬 Evolutionary Odyssey: 46億年の生命進化シミュレーション

荒れ狂う原始の海から始まり、過酷な大絶滅の試練を乗り越え、未来の究極生命体「ネオ・サピエンス」への到達を目指す、2Dリアルタイム生命進化アクションシミュレーションゲームです。

美しいサイバーネオン調のビジュアル、滑らかに動くインラインSVG風景アート、そしてお好みのAI生成画像（DALL-E 3やMidjourneyなど）をシームレスに統合できる「ハイブリッド画像読み込み＆安全なSVGフォールバック」設計を特徴としています。

---


## ✨ コア機能と特徴

- 🧬 **5つの進化時代（全5世代）の体験**:
  - **冥王代・原始生命**（単細胞）から**原生代、古生代、中生代、新生代**にわたる生命形態のダイナミックな変遷を、美麗な発光（バイオルミネッセンス）イラストで表現。
- 🦖 **遺伝子変異と自機パーツの動的描画**:
  - 「べん毛のうねり」「ヒレ」「硬質シェル」「トゲ反撃」「熱・寒冷耐性」などの変異をアンロックすると、パラメータが向上するだけでなく、ゲーム画面上の自機ビジュアルに自動的にパーツが合成され、リアルタイムでアニメーションします。
- 🌋 **6大絶滅イベントと「動くSVG」シネマティック風景**:
  - 大酸化破局、全地球凍結、カンブリア捕食者爆発、火山大爆発、隕石衝突、大氷河期の発生時にゲームが一時停止し、破滅の様子を描いた大迫力の動くSVG風景アートと生存遺伝子の適応判定バッジを表示します。
- 📸 **外部精密画像のハイブリッド読み込み（堅牢設計）**:
  - AI等で生成したお好みの精密画像（PNG）を `public/images/events/` に置くと優先ロードされ、画像がない（またはロード失敗）場合は、自動的に上記の「美麗な動くSVG風景」やデフォルトアセットへ安全にフォールバックされます。画面が真っ黒になったりフリーズしたりすることは100%ありません。
- 👑 **ネオ・サピエンスへの究極進化**:
  - 全変異をアンロックした瞬間に「究極進化（ゲームクリア）」へとシームレスに遷移。横長ワイドスクリーン画像に、深呼吸するようなパララックス・ズーム浮遊アニメーション（`clear-image-zoom-float`）を付与した、最高級のクリアビジュアルが表示されます。
- 📖 **充実の進化図鑑（Evolution Gallery）**:
  - 到達した最高時代に応じて徐々にロックが解除されるプレミアムな図鑑画面を搭載。
- 🚑 **その場での復活（コンティニュー機能）**:
  - 絶滅してゲームオーバーになった際、HPを全回復し、3.5秒間の無敵状態と周囲の敵を安全位置へ退避させるセーフガードを適用した状態で、その場からプレイを続行できる親切設計。

## 🛠️ ローカル開発環境でのセットアップ手順

本作は **Vite + React + TypeScript** で構築されています。

### 1. 必要要件
- **Node.js**: `v18.0.0` 以上推奨
- **npm**: `v9.0.0` 以上推奨

### 2. セットアップの手順
リポジトリをクローンまたはフォルダを開いた後、以下のコマンドを実行します。

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

### 3. スマートフォン等からのローカルアクセス方法
同じWi-Fiルーター（ローカルネットワーク）内にあるスマートフォンやタブレットからアクセスしてプレイしたい場合は、`--host` オプションを付与して起動します。

```bash
npm run dev -- --host
```
起動後、表示される `http://[あなたのローカルIP]:5173` （例: `http://192.168.3.152:5173`）にスマートフォンのブラウザからアクセスします。

## 🎮 ゲームの遊び方 (Gameplay Guide)

### 🕹️ 操縦方法 (Controls)
- **PC操作**:
  - マウスカーソルを動かす（自機がマウスの方向へ引き寄せられるように追従します）
  - キーボード操作: `W` `A` `S` `D` または `矢印キー` （WASDで繊細な移動が可能です）
- **スマートフォン/タブレット操作**:
  - 画面をタッチしてドラッグ（直感的に指の動きに追従して遊泳します）

### 📝 基本ルール
1. **エネルギー（栄養）の回収**:
   - 画面内に浮かんでいる「光るアミノ酸・栄養粒子」を食べると、スコアと進化素材（ミュータジェン）を獲得できます。
2. **生命力（HP）の維持**:
   - 浮遊する「敵対生物」や「障害物質（有毒泡やマグマ球体）」に接触するとダメージを受けます。
   - 体力が0になると「絶滅（ゲームオーバー）」となりますが、「その場で復活する」メニューでHPを全回復して再開することも可能です。
3. **進化モーダルでの遺伝子変異**:
   - 時代ゲージがMAXになると、画面上部の「🧬 変異進化」ボタンがアクティブになります。クリックして進化モーダルを開き、欲しい遺伝子コードをアンロックしましょう。
4. **大絶滅イベントの克服**:
   - 時代が進むごとに、破滅的な「大絶滅イベント」が発生します。
   - 各イベントを無傷で乗り越えるには、**特定の生存適応能力（遺伝子変異）**をあらかじめ獲得しておく必要があります（例: 全地球凍結なら「防寒・羽毛化」、隕石衝突なら「地下潜行/耐爆」など）。適応していない場合、激しいスリップダメージを受けるため、事前の進化計画が重要です。
5. **最終ゴール**:
   - 新生代まで生き残り、すべての進化遺伝子を網羅して、宇宙唯一無二の「究極生命体：ネオ・サピエンス」を完成させてください！

## 🚀 GitHub Pages への自動デプロイ方法

最もスマートで簡単な方法は、**GitHub Actions** を使用した自動デプロイです。リポジトリに push するだけで、ビルドから公開まで自動で行われます。

### 1. GitHub Actions の設定ファイル作成
プロジェクトのルートに `.github/workflows/deploy.yml` ファイルを作成し、以下のコードを記述します。

```yaml
name: Deploy to GitHub Pages

on:
  push:
    branches: [ main ] # メインブランチにpushされたら実行

permissions:
  contents: read
  pages: write
  id-token: write

concurrency:
  group: 'pages'
  cancel-in-progress: true

jobs:
  deploy:
    environment:
      name: github-pages
      url: ${{ steps.deployment.outputs.page_url }}
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Set up Node.js
        uses: actions/setup-node@v4
        with:
          node-version: 20
          cache: npm

      - name: Install dependencies
        run: npm ci

      - name: Build
        run: npm run build

      - name: Upload artifact
        uses: actions/upload-pages-artifact@v3
        with:
          path: ./dist

      - name: Deploy to GitHub Pages
        id: deployment
        uses: actions/deploy-pages@v4
```

### 2. GitHub リポジトリの設定
GitHub 上でリポジトリの設定（Settings）を開きます：
1. **Pages** タブを開きます。
2. **Build and deployment** > **Source** を 「**GitHub Actions**」 に設定します。
3. あとは、コードを `main` ブランチにプッシュするだけで、数分後に `https://[ユーザー名].github.io/[リポジトリ名]/` にてゲームが世界に公開されます！
